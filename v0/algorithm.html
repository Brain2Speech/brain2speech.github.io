Our algorithms are split into 2 main categories: Speech Synthesis and Signal Processing.
You can learn more about the algorithms that are used in our research and run online examples.

The algorithms for Brain2Speech are freely available under a 2-clause BSD license. Please see the LICENSE file for details.
All versions of our code are available for download on GitHub.
In order to get the best experience, please make sure you run the algorithms on a machine that meets the basic requirements:

  • Python 3.6.0
  • Supported systems: 64 bit versions of Linux, MacOS, Windows
  • and other specific requirements (listed in the README for each algorithm group)
